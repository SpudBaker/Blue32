32 Blue by default will use an open source MySql database (or a derivative) located on the Amazon cloud.


32 Blue utilises a Single Page Architecture (SPA) built with the Angular framework allowing for rich, responsive, and complex web applications.

Presentation is kept separate from business logic and re-useable RESTful data services are utilised to connect to the database.

Node.js is the dominant open source Javascript runtime for servers and has become an industry standard for hosting web applications and services.
A secure bespoked password mechanism with users defined into roles with individual permissions can built into the applicaton.

In addition with the web server running in Node.js it is possible to integrate with 3rd party authentication services like Google or Facebook.
Applications are hosted on the Amazon cloud allowing for high availability and performance

This removes the requirement for any local software or hardware making the solution low cost and low maintenance.
